---
title: 앱 라우터
description: 레이아웃, 서버 컴포넌트, 서스펜스 등을 포함한 Next.js와 React의 최신 기능을 새로운 앱 라우터와 함께 사용해보세요.
---

앱 라우터는 React의 최신 기능을 사용하여 애플리케이션을 구축하는 새로운 방식입니다. Next.js에 익숙하다면 앱 라우터가 기존의 파일 시스템 기반 라우터인 [페이지 라우터](/docs/pages)의 자연스러운 발전임을 알 수 있습니다.

새로운 애플리케이션에 대해서는 앱 라우터를 사용하는 것을 권장합니다. 기존 애플리케이션에 대해서는 [앱 라우터로 점진적으로 마이그레이션](/docs/app/building-your-application/upgrading/app-router-migration) 할 수 있습니다.

해당 문서 섹션은 앱 라우터에서 사용할 수 있는 기능들을 담고 있습니다.
